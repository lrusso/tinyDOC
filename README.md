# TinyDOC Editor

TinyDOC Editor is a HTML5 Rich Text Editor. The idea of this project is to provide a lightweight text editor with simple and easy-to-use features.

![alt screenshot](https://raw.githubusercontent.com/lrusso/tinyDOC/master/tinyDOC.png)

## Web

https://lrusso.github.io/tinyDOC/tinyDOC.htm

## Calc Button Sample 1 - Single Line Evaluation

```
lorem ipsum (5+4-3)*2/1 lorem ipsum
```
Select (5+4-3)*2/1, then click in the Calc Button, and the selected text will be replaced with the result.

```
lorem ipsum 12 lorem ipsum
```

## Calc Button Sample 2 - Multi Line Addition Operation

```
$500
$300
$50
```

Select all, then click in the Calc Button, and the selected text will be replaced with the result.

```
850
```
